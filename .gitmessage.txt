# Type에는 다음과 같은 종류가 올 수 있다.
# Feat: 새로운 기능
# New : 완전히 새로운 기능
# Improve : 기존 기능에서 수정, 요구사항 변경으로 인한 수정
# Fix: 버그 수정
# Docs: 문서 변경
# Style: 코드 포맷팅, 세미콜론 추가 등(코드 변경 없음)
# Refactor: 프로덕션 코드 리팩토링
# Test: 테스트, 리팩토링 작업, 프로덕션 코드 변경 없음
# Chore: 빌드작업 업데이트, 패키지 관리자 구성 변경 등 프로덕션 코드 변경 없음
# Release : 릴리즈 배포 버전 기입용
